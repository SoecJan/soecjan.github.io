Detailed Description

This interface specifies functionality for ProductProcessorBehaviour, CityPlaceable and Warehouse. It's purpose is to make ProductStorages accessible to display to the user, like ProductProcessorView, CityView and StorageContainerView. Transportation logic heavily relies on this functionality in TransportVehicle.
